Linux Cheat Sheet

Navigation

  • pwdPrints the full path of your current working directory.
  • lsLists files and folders in the current directory.
  • ls -laLists all files including hidden ones, with permissions and ownership.
  • cd <dir>Moves you into the specified directory.
  • cd ..Moves one level up in the directory tree.
  • cd -Switches back to your previous directory.

File & Directory Management

  • touch fileCreates an empty file or updates its timestamp.
  • mkdir dirCreates a new directory.
  • mkdir -p a/b/cCreates nested directories in one command.
  • cp src destCopies a file to a new location.
  • cp -r src destRecursively copies a directory and its contents.
  • mv src destMoves or renames files/directories.
  • rm fileDeletes a file.
  • rm -r dirDeletes a directory and its contents.
  • rm -rf dirForce-deletes a directory without prompts (dangerous).

Viewing & Editing Files

  • cat fileDisplays the entire file contents.
  • tac fileDisplays file contents in reverse order.
  • head fileShows the first 10 lines of a file.
  • tail fileShows the last 10 lines of a file.
  • tail -f fileStreams new lines as they’re added (useful for logs).
  • nano fileOpens a simple terminal text editor.
  • vim fileOpens a powerful text editor with advanced features.

Search

  • grep "text" fileSearches for matching text inside a file.
  • grep -r "text" dirRecursively searches through directories.
  • grep -i "text"Performs a case-insensitive search.
  • grep -n "text"Shows line numbers for each match.
  • find . -name "*.txt"Finds files by name pattern.
  • find . -type f -size +10MFinds files larger than 10MB.

System Information

  • uname -aShows kernel version and system architecture.
  • hostnamectlDisplays system hostname and OS details.
  • topLive view of running processes and resource usage.
  • htopEnhanced version of top with better UI.
  • df -hShows disk usage in human-readable format.
  • du -sh *Shows size of each file/folder in the current directory.
  • free -hShows RAM usage.
  • uptimeShows system uptime and load averages.

Process Management

  • ps auxLists all running processes with detailed info.
  • kill <pid>Sends a termination signal to a process.
  • kill -9 <pid>Force-kills a process immediately.
  • pkill nameKills processes by name instead of PID.
  • jobsLists background jobs in your shell.
  • bgResumes a job in the background.
  • fgBrings a background job to the foreground.

Permissions & Ownership

  • chmod 755 fileOwner can read/write/execute; others can read/execute.
  • chmod 644 fileOwner can read/write; others can only read.
  • chown user:group fileChanges file owner and group.
  • sudo commandRuns a command with root privileges.

Package Management

Debian / Ubuntu (APT)

  • sudo apt updateRefreshes package lists.
  • sudo apt upgradeInstalls available updates.
  • sudo apt install pkgInstalls a package.
  • sudo apt remove pkgRemoves a package.

RHEL / CentOS / Fedora (DNF)

  • sudo dnf install pkgInstalls a package.
  • sudo dnf updateUpdates all packages.
  • sudo dnf remove pkgRemoves a package.

Arch (Pacman)

  • sudo pacman -S pkgInstalls a package.
  • sudo pacman -R pkgRemoves a package.
  • sudo pacman -SyuPerforms a full system update.

Networking

  • ip aShows all network interfaces and IP addresses.
  • ip rDisplays routing table.
  • ping hostTests network connectivity.
  • curl urlFetches data from a URL.
  • wget urlDownloads files from the internet.
  • ss -tulpnLists open ports and listening services.
  • ssh user@hostConnects to a remote machine via SSH.

Compression & Archiving

  • tar -cvf file.tar dirCreates a tar archive from a directory.
  • tar -xvf file.tarExtracts a tar archive.
  • tar -czvf file.tar.gz dirCreates a compressed gzip tarball.
  • tar -xzvf file.tar.gzExtracts a gzip tarball.
  • zip file.zip fileCompresses files into a zip archive.
  • unzip file.zipExtracts a zip archive.

User & Group Management

  • sudo adduser nameCreates a new user.
  • sudo deluser nameDeletes a user.
  • sudo passwd nameChanges a user’s password.
  • sudo groupadd groupCreates a new group.
  • sudo usermod -aG group userAdds a user to a group.

Services (systemd)

  • systemctl status serviceShows service status.
  • systemctl start serviceStarts a service.
  • systemctl stop serviceStops a service.
  • systemctl restart serviceRestarts a service.
  • systemctl enable serviceEnables a service at boot.
  • systemctl disable serviceDisables a service at boot.

Disk & Partition Tools

  • lsblkLists block devices (drives, partitions).
  • blkidShows filesystem UUIDs.
  • mount /dev/sdX /mntMounts a device to a directory.
  • umount /mntUnmounts a device.
  • fdisk -lLists partitions on all drives.
  • sudo fdisk /dev/sdXOpens partition editor for a device.

Docker Essentials

  • docker psShows running containers.
  • docker ps -aShows all containers (running and stopped).
  • docker imagesLists available images.
  • docker run -it imageRuns a container interactively.
  • docker stop idStops a running container.
  • docker rm idRemoves a container.
  • docker rmi idRemoves an image.

Git Basics

  • git initCreates a new repository.
  • git clone urlDownloads a remote repository.
  • git statusShows changes and repository state.
  • git add .Stages all changes.
  • git commit -m "msg"Saves staged changes with a message.
  • git pushUploads commits to the remote repository.
  • git pullFetches and merges remote changes.

Shell Tricks

  • historyShows your command history.
  • !!Repeats your last command.
  • !123Runs command number 123 from history.
  • Ctrl + RSearches your history interactively.
  • alias ll="ls -la"Creates a shortcut alias for a command.
